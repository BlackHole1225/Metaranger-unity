Sniper Rifle - FPS Microgame Add-On
===================================

- "FPS Microgame" Version 3.0.0 available from the Unity Hub.
- In the Unity Hub, select the "Projects" tab, click the "New Project" button, from the list of Templates, select the "FPS Microgame".
- You can find more information about this Microgame Add-On here: http://u3d.as/1Lad
